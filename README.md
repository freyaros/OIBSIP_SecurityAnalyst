## Project Overview
This repository contains the tasks completed during my internship at **Oasis Infobyte**. The projects focus on network security, traffic analysis, and comprehensive research into modern cybersecurity threats and social engineering.

## General Requirements & Tools
To replicate or review these tasks, the following tools and environments were utilized:
* **Operating System:** Windows
* **Networking Tools:** Nmap (Network Mapper), Zenmap (GUI), Wireshark (Packet Analyzer)
* **Documentation:** LaTeX (Overleaf)
* **Platform:** GitHub for Version Control

### Completed Tasks
1. [Task 1: Network Scanning with Nmap](./Task1-Nmap)
   * **Objective:** Identify open ports, services, and firewall configurations on a target system.
   * **Key Skills:** OS Fingerprinting, Service Version Detection, Zenmap Topology Mapping.
   * **Outcome:** Successfully generated a detailed security profile of the target environment.
2. [Task 8: Traffic Analysis with Wireshark](./Task8-Wireshark)
   * **Objective:** Capture live packets and analyze HTTP traffic to identify plain-text data vulnerabilities.
   * **Key Skills:** Packet Sniffing, Display Filters, Protocol Analysis (TCP/HTTP).
   * **Outcome:** Demonstrated the security risks of unencrypted traffic and the importance of TLS/SSL.
5. [Task 4: Network Security Threats Report](./Task4-Network_Security_Threats)
   * **Objective:** In-depth research on Denial of Service (DoS), Man-in-the-Middle (MITM), and Spoofing attacks.
   * **Deliverables:** Detailed LaTeX-formatted PDF and Markdown report.
6. [Task 5: Social Engineering Report](./Task5-Social-Engineering)
   * **Objective:** Analysis of psychological manipulation techniques and defense-in-depth frameworks.
   * **Deliverables:** Comprehensive report including the MGM Resorts (2023) Case Study.

## Learning Experience

1. Conduct passive and active reconnaissance effectively.
